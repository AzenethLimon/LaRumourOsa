#DIFERRumorosa

El front-end de este proyecto fue desarrollado utilizando Angular CLI y Materialize CSS, por lo que se requiere tener instalado Angular en el ordenador.

#ServidorDIFER

Para correr el front-end, es necesario abrir tu navegador e insertar en el buscador `http://localhost:4200/`, asimismo agregar 'ng serve' desde la terminal de Visual Studio Code o Command Prompt con el folder abierto.

#Referencias
'ng serve' = correr servidor 
'ng generate component component-name' = agregar nuevo componente
'ng help' = ayuda
'cd folder_name' = abrir folder

#BackEndRumorosa

El BackEnd de este proyecto fue hecho con Node.js, se necesita POSTMAN para utilizar las rutas. El resto se encuentra dentro de la carpeta.
Para correr el BackEnd, se ocupa abrir el folder en Visual Studio Code, ingresar 'ng serve' en la terminal y abrir en el navegador 'http://localhost:3000/'.
